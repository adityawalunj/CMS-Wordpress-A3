Description -

Increase Upload File Size Limit is a Wordpress plugin 
designed to increase the upload limit for files 
from 2mb to a staggering 655 GB in the Wordpress media uploader.
The plugin uses a combination of Ajax and PhP to run smoothly and securely.


How to use the plugin -

1. Download and install the plugin Increase Upload File Size Limit on Wordpress.
2. Activate the plugin
3. Simply go to media uploader to upload big files.
